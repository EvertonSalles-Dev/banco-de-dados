SI Escolar - Sistema Escolar (HTML5 + CSS3 + JavaScript )
================================================================

Como testar:
1. Abra a pasta "SistemaEscolar" no VS Code.
2. Instale a extensão "Live Server" (ou similar).
3. Clique com o botão direito em index.html e escolha "Open with Live Server".
   (Não é obrigatório abrir via servidor, mas é recomendado, pois alguns
   navegadores restringem o carregamento de scripts locais com file://.
   Se preferir, pode também abrir o index.html diretamente clicando duas vezes.)

Contas de demonstração (já pré-cadastradas no LocalStorage):
- Aluno:         aluno@escola.com     / aluno123
- Professor:     professor@escola.com / prof123
- Administrador: admin@escola.com     / admin123

O "banco de dados" é o LocalStorage do seu navegador. Cada navegador/perfil
terá seus próprios dados. Para reiniciar os dados, use a opção
"Limpar banco de dados" dentro do painel do Administrador > Backup,
ou limpe manualmente o LocalStorage do site pelas ferramentas do navegador.
